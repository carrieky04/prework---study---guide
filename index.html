HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The section element allows us to organize the content on our page
  • You can have up to six headers
  • There are unordered and ordered lists

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The box-shadow adds shadow effects around an element's frame.
  • The period preceding "card" designates this as a class selector.
  • Assigning colors in CSS can be done using semantics or a hexadecimal code.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git clone: downloads existing source code from a remote repository
  • git add file: adds a single file.
  • git add -A: adds all the files at once.
  • Important: the git command doesn't change the repo and the changes are not saved until we use git commit
  • git commit -m commit message: saves your changes, but only locally
  • git push GitHub: uploads your commits to the remote repo
  • git push -u origin GitHub: if branch is new, this will upload the branch to repo
  • git pull origin main: sync our local branch with remote branch in GitHub

JavaScript

  • A variable is a name container that allows us to store data in our code.
  • To declare a varibale we use the "var" keyword. Then give the var a name.
  • Use the = to assign the specific data that we want stored in our var.
  • Control flow is the order in which a computer executes code in a file.
  • Using conditional statements such as if statements allows us to interrupt control flow.
  • If condition is truthy, then the computer will do whatever action we ask it to do in between the curly brakets.
  • An array is a group of related data in a variable.
  • A "for" loop uses the predictable pattern of indices to perform a task on all the items in an array by allowing a single code block to be executed over and over.
  • A "for" loop contains 3 statements:
    • a starting point: var = 0
    • the condition: x < shapes.length
    • x++
  • Fuctions are the building blocks of JavaScript. They define a block of code that performs a specific task.
    • To execute a function, you must call it in your code.
    • It is executed from top to bottom, so order matters!
  • Use Google DevTools console in order to see JavaScript running.
